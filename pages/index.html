We're a small growing community that uses our own open-source node.js server (GitHub) to post images and chats to each other over different channels and "conversations".

The server is open to contributions from anyone willing to help out.

New conversations (the LiveChan version of threads) can be made by setting the second field (General by default), or you can join an existing conversation by choosing one from the list on the right once you've joined a board.

Hide this message to join the "All" channel and watch chat messages posted in realtime, or choose a channel from the dropdown box on the top left to start chatting.

Please note there is a strict 7 second cool down time shared through all boards for new chat messages. New conversations have a 50 second cool down time to help prevent spamming.
